# 90秒 / Just 90s

> 陪你度过最容易失控的那90秒

基于"90秒情绪生理周期"理论的极简情绪陪伴Web应用。

## 产品特点

- **零学习成本**：打开即用，无需说明
- **零选择压力**：不需要选择模式或情绪类型
- **身体优先**：先稳神经系统，再谈意义
- **短即正义**：核心体验仅90秒
- **不留痕**：不记录、不分析、不评估

## 技术栈

- **框架**：Vite + React + TypeScript
- **样式**：Tailwind CSS
- **PWA**：支持离线使用，可添加到主屏幕
- **音频**：Web Audio API（可选）

## 核心流程

90秒陪伴流程分为4个阶段：

1. **确认与允许（0-10秒）**
   - 提示："你现在在情绪反应中，这很正常。"

2. **身体锚定（10-60秒）**
   - 呼吸引导：吸气4秒，呼气6秒
   - 柔和的视觉动画
   - 可选的环境音

3. **等待情绪回落（60-90秒）**
   - 提示："这股感觉正在自然下降"
   - 缓慢消散的动画

4. **选择权回归**
   - "我可以继续一天"或"我还需要一点陪伴"

## PWA 安装

### iOS Safari

1. 在 Safari 中打开应用
2. 点击分享按钮
3. 选择"添加到主屏幕"

### Android Chrome

1. 在 Chrome 中打开应用
2. 点击菜单按钮
3. 选择"添加到主屏幕"或"安装应用"

## 设计原则

### 明确不做

- ❌ 情绪分类
- ❌ 日记/记录
- ❌ 数据统计
- ❌ AI分析/说教
- ❌ 打分/成就系统

### 原因

这些功能会把用户重新拉回"头脑层面"，而我们专注于陪伴用户安全地穿过情绪浪潮。

## 移动端优化

- 单列布局，大按钮设计
- 所有交互位于拇指热区
- 支持深色模式
- 处理iOS自动锁屏
- 响应式动画帧率（30-45 fps）

## 浏览器支持

- ✅ iOS Safari 14+
- ✅ Android Chrome 90+
- ✅ Desktop Chrome/Firefox/Safari（次要支持）
